This page lists and describes the java files in your elevatorcontrol package.  Update this document to reflect your implementation and add your java files to the elevatorcontrol folder.

Note:  In order for your portfolio to be complete, the code files in the elevatorcontrol folder must compiled and run if copied into the elevatorcontrol package (overwriting any duplicates) of a clean version of the latest required simulator release.

Controller files

  • CarButtonControl.java - Controller that handles CarCalls and CarLight
  • DoorControl.java - Controller that manages the DoorMotors and opens and closes the doors
  • DriveControl.java - Controller for the Drive that handles the movement of the elevator
  • HallButtonControl.java - Controller that handles hall button presses
  • LanternControl.java - Controller that manages CarLanern
  • CarPositionControl.java - Controller that handles the car position indicator
  • Dispatcher.java - Controller that handles desired floors

Translator files

Links and descriptions to all message translators used by the controllers.
  • DesiredDwellCanTranslator.java - Translator for the mDesiredDwell network message from the Dispatcher
  • DesiredFloorCanPayloadTranslator.java - Translator for mDesiredFloor network message from the Dispatcher
  • DriveSpeedCanPayloadTranslator.java - Translator for the mDriveSpeed message that comes from the driveSpeed sensor
  • CarPositionCanPayloadTranslator.java - Translator for the mCarPosition message that comes from the car position sensor
  • DriveCommandCanPayloadTranslator.java - Translator for the mDrive message that comes from the drive controller.
  • CarCallCanPayloadTranslator.java - Translator for the mCarCall message that comes from various car call controllers
  • HallCallCanPayloadTranslator.java - Translator for the mHalCall message that comes from various hall call controllers

Utility files

  • Proj7RuntimeMonitor.java - Runtime monitor for project 7 that checks for wasted openings, overweight instances, and time spend reversing doors.
  • Proj11RuntimeMonitor.java - Runtime Monitor for project 11 that checks that requirements are not broken.
  • MessageDictionary.java - Collection of network message encodings..
  • Utility.java - A file of various utility functions for determining current floor.
